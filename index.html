About me

Who I Am

My name is Erica Piano, and I am a certified fullstack web developer in Central New Jersey. I have a special interest in frontend development, most specifically styling sites using CSS, Bootstrap and SASS. I am currently seeking a full time developer role.

What I do

I take the most pride in creating premium content while keeping a clean code base. I am able to create a full web application with both frontend and server-side functionality. I typically utilize HTML, CSS, JS, and React on the from-end and node.js and express on the backend. For data storage, I prefer MongoDB- coupled with mongoose- however I am proficient with MySQL and SQL as well.

My tools


REACT

JS is a powerful language and has afforded me the ability to learn many frontend languages including ES6, Vanilla JavaScript, and JQuery. I most enjoy writing my frontend in React, as it allows for reusable components and fast production which delivers demanding frontend solutions.

NODE.JS

I use my JavaScript knowledge to power my backend using Node.js. I incorporate npm packages such as cheerio, axios, express, mongoose, and sequelize to make the server side work smoothly.




MongoDB

I use MonogDB to build and maintain the databases of my applications. Through node I use mongoose to communicate with the server. Similarly I have built databases using MySQL, using sequelize to communicate to the server.
